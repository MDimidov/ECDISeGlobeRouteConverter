# <img src="https://github.com/MDimidov/ECDISeGlobeRouteConverter/blob/main/ECDIS%20eGloebe%20-%20RouteConverter/eGlobe-min.ico" width="32px" height="32px"> ECDIS eGlobe Route Converter

Welcome to **ECDIS eGlobe Route Converter Windows Form**, a user-friendly voyage deserializer and passage plan generator, built using *C#* and *Windows Forms*. This tool simplifies maritime route conversion and enhances navigation planning efficiency.

---

## üåü Features
- Deserialize voyage data from `.rte` files generated by ECDIS systems.
- Convert and export passage plans for future use.
- User-friendly interface with interactive controls and real-time feedback.
